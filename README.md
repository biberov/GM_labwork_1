## Инструменты и технологии
* **Среда разработки (IDE):** Microsoft Visual Studio 2026
* **Платформа:** .NET (Windows Forms)
* **Язык программирования:** C#

## Основные функции приложения

### 1. Ввод данных
Пользователь задает 6 параметров в специальных текстовых полях:
* Координаты центра прямоугольника (x_c, y_c)
* Высота прямоугольника (h)
* Ширина прямоугольника (w)
* Координаты внешней точки (x, y)

### 2. Построение фигур
* **Прямоугольник** строится по четырем угловым точкам, рассчитанным на основе центра, высоты и ширины
* **Отрезок** строится от центра к внешней точке

### 3. Вычисление пересечения
* Реализован геометрический алгоритм, проверяющий пересечение луча (от центра к внешней точке) с каждой из четырех сторон прямоугольника
* Алгоритм использует параметрическое уравнение прямой и находит ближайшую к центру точку пересечения

### 4. Визуализация
* **Прямоугольник:** отображается синей линией
* **Отрезок:** отображается красной линией
* **Точки:**
  * Центр прямоугольника отмечен черным кружком с подписью координат
  * Внешняя точка отмечена оранжевой звездой с подписью координат
  * Найденная точка пересечения (если существует) отмечена зеленым кружком с подписью координат

### 5. Автомасштабирование
График автоматически подстраивает отображаемую область так, чтобы все элементы (прямоугольник, отрезок и точки) были видны с оптимальными отступами (20% от размера фигур).

### 6. Обработка особых случаев
* Если внешняя точка лежит точно на границе прямоугольника, программа уведомляет об этом пользователя
* Если отрезок не пересекает прямоугольник, программа очищает график и выводит соответствующее сообщение

## Структура кода (ключевые методы)

| Метод | Описание |
|-------|----------|
| `SetupChart()` | Инициализирует график, настраивает оси и создает серии данных для отображения всех элементов |
| `DrawRectangle()` | Вычисляет углы прямоугольника и добавляет их в соответствующую серию графика |
| `DrawLine()` | Добавляет точки начала и конца отрезка в серию графика |
| `FindIntersection()` | Ядро программы. Содержит математическую логику поиска точки пересечения отрезка с прямоугольником |
| `AutoScaleChart()` | Автоматически подбирает масштаб осей для комфортного отображения всех объектов |
| `buttonDraw_Click()` | Главный обработчик события, запускающий весь процесс: считывание данных, очистку, построение, поиск и обновление графика |
